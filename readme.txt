== Description ==

You can now view your attachments in your post on your wordpress blog with views and clicks count analytic.

Once installed as a WordPress plugin, All attachment you attached it to post will than automatically appear on selected post, to install just active it.

== Installation ==

To install, simple extract the 'attachment_analytic' folder into your 'wp-content/plugins/' directory. Once extracted, you must activate the plugin within the WordPress Site Admin 'Plugins' section.
